Christmas GIFt

MISC Easy

Description

Here is your christmas GIFt from santa! Just open and wait for it..

Solution

  1. Initial Analysis:
    1. Given a GIF file with an animated Christmas present
    2. Challenge name has "GIF" capitalized, hinting at GIF manipulation
    3. Description suggests waiting/looking through the entire animation
  2. Using Stegsolve:
    1. Tool used: Stegsolve for GIF analysis
    2. Used the Frame Browser feature to examine individual frames
    3. The GIF contained 1402 total frames
    4. The flag was hidden in the very last frame (1402 of 1402)
  3. Flag Discovery:
    1. Found in frame 1402: Image of a present with the flag visible
    2. Flag format: wgmy{1eaa6da7b7f5df6f7c0381c8f23af4d3}
    3. Flag was clearly visible in plain text on the final frame

Tools Used

  • Stegsolve
  • Frame Browser functionality

Key Takeaways

  • Always check all frames in a GIF file
  • Pay attention to challenge descriptions - "wait for it" was a hint
  • Sometimes the simplest solution (checking the last frame) is correct
  • The capitalization in "GIFt" was a subtle hint about GIF analysis

Tips for Similar Challenges

  • Check beginning and end frames of GIFs
  • Use frame browsing tools for animated files
  • Pay attention to challenge names and descriptions for hints
  • Don't overcomplicate the solution - sometimes flags are hidden in plain sight
